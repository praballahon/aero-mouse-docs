# Aero Mouse
 
Aero Mouse transforms your Android device into a wireless Bluetooth mouse, keyboard, and media controller for PCs, Macs, TVs, tablets, and other compatible devices.
 
## Features
 
- Wireless Bluetooth mouse control
- Keyboard input support
- Media playback controls
- Works with Windows PCs, macOS devices, TVs, tablets, and more
- Lightweight and simple UI
- No advertisements
- No user tracking
- No analytics collection
 
## Permissions
 
### Bluetooth
Required for:
- Discovering nearby devices
- Pairing with supported devices
- Establishing Bluetooth connections
- Sending mouse, keyboard, and media control inputs
 
### Location
Android requires location permission for Bluetooth device discovery and pairing.
 
Aero Mouse does not collect or track your physical location.
 
## Privacy
 
Aero Mouse does not collect, store, transmit, or share any personal user data.
